# TBW AI PREMIUM NAVIGATOR – Web / APK Shell

Ovo je minimalni, ali funkcionalni frontend + struktura za TBW AI PREMIUM
NAVIGATOR. Frontend koristi `/api/tbw` backend rutu na Vercelu.

- `index.html` – glavna početna stranica (layout kao na mockupu)
- `style.css` – tamni premium UI
- `app.js` – spajanje na backend rute
- `backend/tbw.js` – Vercel serverless funkcija (ubaci svoju punu verziju)
- `backend/env.template` – popis env varijabli
